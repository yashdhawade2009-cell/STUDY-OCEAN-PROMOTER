▼ DEPLOYMENT GUIDE

How to publish new promotions — no backend needed

  1. Locate the /promotions/ folder in your project root (or the deployed site's public folder).
  2. Copy your image file into the folder. Name it using the pattern:
    promo1.jpg  •  promo2.png  •  promo3.webp
  3. The file name number determines the order. Start from promo1 and increment sequentially.
  4. Reload the homepage — the image will appear automatically in the live gallery.
  5. To remove a promotion, delete the file from the /promotions/ folder and redeploy.
⚠ IMPORTANT: This platform uses static hosting. File uploads from the browser UI are not saved to the server automatically — you must manually place files in the promotions folder and redeploy or commit via your hosting platform (Vercel, Netlify, etc.).
